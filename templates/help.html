Step 1: Get Your Amazon Associate ID

1. Visit affiliate-program.amazon.com

2. Sign up for a free Amazon Associates account

3. Get your Associate ID (looks like "yourname-20")

4. Enter it in your dashboard setup

Step 2: Setup Discord (Recommended)

1. Create a Discord server or use an existing one

2. Create a webhook: Server Settings ‚Üí Integrations ‚Üí Webhooks

3. Copy the webhook URL and paste it in your platform setup

Webhook URL format: https://discord.com/api/webhooks/...

Step 3: Setup Telegram (Optional)

1. Create a bot by messaging @BotFather on Telegram

2. Use command: /newbot and follow instructions

3. Get your bot token and chat ID

4. Enter both in your platform setup

Step 4: Setup Slack (Optional)

1. Create a Slack app at api.slack.com

2. Add bot token scopes: chat:write, files:write

3. Install app to your workspace

4. Copy bot token and channel ID to your setup
